## 特点
- 自动获取番剧更新并下载至对应网盘
- 通过PushPlus, Telegram等渠道发送更新通知
- 支持PikPak, 115等离线下载
- AI自主分析资源名，并结合搜索TMDB数据库，获取准确的番剧名，季度，集数来实现重命名
- 拥有你的AI智能下载助手，可挂载到telegrambot上，通过自然语言的方式让ai帮你搜索并下载资源

## 准备工作 
1. 请自行参照[Openlist](https://doc.oplist.org/guide)项目文档部署Openlist，并搭建好离线下载
2. 准备好RSS订阅连接
